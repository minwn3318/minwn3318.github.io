# C 언어 - 서식 지정자   
### "서식 지정자" 란?   
```
: 프로그래밍 언어에서 특정 변수 타입에 알맞게 출력하거나 입력하기 위해 사용되는 기호를 가리킨다.   
```

       
C 언어에서 입/출력을 구현할려면 기본적으로 scanf와 printf가 사용된다.   
자바나 C++의 경우 바로 변수를 사용하면 자동적으로 타입에 맞게 입출력이 이루어지만,    
순수 C 언어의 경우 개발자가 타입을 지정하여야 그에 알맞는 타입 입출력이 이루어진다   

해당 포스트에서는 C언어 입출력에서 사용되는 서식 지정자를 알아보도록 하겠다.   

    
   
### 1. 진수 포맷팅     
   
| 서식 지정자 | 의미              | 사용 자료형         | 
| ------ | --------------- | -------------- |
| `%d`   | 부호 있는 10진수    | `int`          | 
| `%u`   | 부호 없는 10진수    | `unsigned int` | 
| `%o`   | 부호 없는 8진수       | `unsigned int` | 
| `%x`   | 부호 없는 16진수, 소문자 | `unsigned int` |
      
### 2. 정수 포맷팅    
   
| 서식 지정자 | 의미              | 사용 자료형         | 
| ------ | --------------- | -------------- |
| `%hhd`  | char를 부호 있는 정수로 변환 | `char`     | 
| `%hhu`  | char를 부호 없는 정수로 변환 | `unsinged char`     | 
| `%hd`  | 부호 있는 정수 짧은 형태 | `short`     | 
| `%hu`  | 부호 없는 정수 짧은 형태 | `unsinged short`     | 
| `%i`  | 부호 있는 정수 형태 | `int`     | 
| `%ld`   | 부호 있는 정수 긴 형태   | `long`          |
| `%lu`  | 부호 없는 정수 긴 형태 | `unsinged long`     | 
| `%lld`   | 부호 있는 정수 가장 긴 형태   | `long long`          |   
| `%llu`  | 부호 있는 정수 가장 짧은 형태 | `unsinged long long`     | 

### 2. 실수 포맷팅    
   
| 서식 지정자 | 의미                    | 사용 자료형            |
| ------ | --------------------- | ----------------- | 
| `%f`   | 일반 실수 표기              | `float`, `double` | 
| `%lf`  | `double` 입력/출력        | `double`          | 
| `%Lf`  | `long double` 입력/출력   | `long double`     | 
| `%e`   | 지수 표기, 소문자            | `double`          | 
| `%g`   | `%f` 또는 `%e` 중 짧은 형태  | `double`          | 
   
- 실수의 경의 자릿수까지만 표시하는 방법으로 `%.2f`가 있다

### 3. 문자 포맷팅    
   
| 서식 지정자 | 의미    | 사용 자료형    | 
| ------ | ----- | --------- | 
| `%c`   | 문자 1개 | `char`    | 
| `%s`   | 문자열   | `char 배열` | 
   
### 4. 주소 포맷팅    
   
| 서식 지정자 | 의미        | 사용 자료형 | 
| ------ | --------- | ------ | 
| `%p`   | 메모리 주소 출력 | 포인터    | 

### 6. 특수출력 포맷팅    
   
| 표현   | 의미        |
| ---- | --------- |
| `\n` | 줄바꿈       |
| `\t` | 탭         |
| `\\` | 역슬래시 출력   |
| `\"` | 큰따옴표 출력   |
| `\'` | 작은따옴표 출력  |
| `%%` | `%` 문자 출력 |
