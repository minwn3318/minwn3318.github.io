# 정보처리기사 - 응용 SW 기초 기술 활용 (SW/HW/DB)  
   
해당 포스트에서는 정보처리기사 개념문제집의 11장 **응용 SW 기초 기술 활용의 DB/HW/SW**를 정리한다   

## SW, HW, DB
### SW 신기술
- 블록체인
> P2P 네트워크르 이용하여 온라인 금융 거래 정보를 온라인 테트워크 참여자의 디지털 장비에 분산저장하는 기술이다

- 매시업
> 다수이 정보원이 제공하는 콘텐츠를 조합하여 하나의 서비스로 제공하는 웹 사이트 또는 애플리케이션을 말한다

- 서비스 지향 아키텍처
> 기버의 소프트웨어 인프라인 정보시스템을공유와 재사용이 가능한 서비스 단위나 컴포넌트 중심으로 구축하는 정보 기술 아키텍처이다

- 디지털 트윈
> 현실속 소프트웨어로 가상화한 모델이다

- 그레이웨어
> 악의적이지 않은 유용한 소프트웨어라고 주장할 수 있지만, 사용자 입장에서는 유용할 수도 있고 악의적이수도 있는 애드웨어

- 양자 암호키 분배
> 양자 통신을 위해 비밀키를 분배하여 관리하느 기술

- 서비스형 소프트웨어
> 소프트웨어의 여러 기능 주에서 사용자가 필요로하는 서비스만 이용할 수 있도록 한 소프트웨어이다

- 시맨틱 웹
> 커퓨터가 사람을 대신하여 정보를 읽고 이해하고 가오하여 새로운 정보를 만들어 낼 수 있도록 이해하기 쉬운 의미를 가진 차세대 지능혀 웹

- 리치 인터넷 애플리케이션
> 신개념 플래시 웹 페이지 제작 기술

---
### HW 신기술
- 앤 스크린
> 다르 단마릭에서 동일한 코넽츠를 자유롭게 이용할 수 있는 서비스
- 신 클라이너트
> 기본적인 메모리만 갖추고 서버와 네트워크로 운용되는 개인용 컴퓨터이다
- 엠디스크
> 한 번의 기록만으로 자료를 영구 보관할 수 있는 광저장장치
- 멤스
> 초정밀 반도체 제조 기술을 바탕으로 센서, 액추에이터 등 기계 구조를 다양한 기술로 미세 가공하여 전기기계적 동작을 할 수있도록 한 초미세 장치이다
- 멤리스터
> 메모리와 레지스터의 합성올 저뉼의 방햐오가 양 등 기존의 경험을 모두 기억하는 특별한 소자
- 트러스트존 기술
> 일반 구영과 보안구역으로 분할하여 관리하는 하드웨어 기바느이 보안기술
- RAID
> 2개 이상의 하드디스크로 디스크 배열을 구성하고 파일을 구성하는 데이터 블록들을 새로 다른 디스크에 분산 저장하거나 다중화하는 저장 기술로 구현된다

---
### DB 신기술
- 하둡
> 오픈 소스를 기바능로 한 분산 커퓨팅 플랫폼
- 맵리듀스
> 대용량 데이터를 분산 처리하기 위한 목적으로개발된 모델
- 데이터 마이닝
> 데이터에 내재된 변수 사이의 상호관계를규명하여 일정한 패턴을 찾아내는 기법
- 타조
> 오픈 소스 기반 분산 커뮤팅 플랫폼인 아파치 하둡 기반의 부산 데이터 웨어하우스 프로젝트이다
- OLAP
> 다차원으로 이루어진 데이터로부터 통계적인 요약정보를 분석하여 의사결정에 활용하는 방식이다
- 브로드 데이터
> 다양한 채널에서 소비자와 상호작요을 통해 생성된것으로 기업 마케팅에 있어 효울적이고 다양한 데이터이며 이전에 사용하지 않거나 알지 못했떤 새로운 데이터나 기존 데이터에 새롱누 가치가더해진 데이터

### 회복 / 병행제어
```
: 데이터베이스가 손사오디었을 때 손상되기 이전의 정상 상태로 복구하는 작업을 가리킨다
```

#### REDO / UNDO
- REDO
```
: 디스크에 저장된 로그를 분석하여 트랜잭션의 시작과 완료에 대한 기록이 있는 트랜잭션들의 작업을 재작업한다
```

- UNDO
```
: 디스크에 저장된 로그를 분석하여 트랜잭션의 시작에 대한 기록은 있지만 완료 기록은 없는 트랜잭션에 대해 모두 취소한다
```

#### 즉각 갱신기법
```
: 트랜잭션이 데이터를 갱신하면 트랜잭션이 부분 완료되기 전이라도 즉시 실제 데이터베이스에 반영하는 방법
```

#### 로킹
```
: 액세스하기 전에 잠금을 요청해서 락이 허락되어야만 그 로킹 단위를 액세스할 수 있도록 하는 기법
```
- 로킹 단위
```
: 병행제어에서 한꺼번에 로킹할 수 있는 객체의 크기를 의미한다
```

#### 타임스탬프 순서
```
: 시간표를 부여하여 부여된 시간에 따라 트랝개션 작어블 수행하는 기법
```
