# 정보처리기사 - 응용 SW 기초 기술 활용 (SW/HW/DB)  
   
해당 포스트에서는 정보처리기사 개념문제집의 11장 **응용 SW 기초 기술 활용의 DB/HW/SW**를 정리한다   

## SW, HW, DB

- 블록체인
- 매시업
- 서비스 지향 아키텍처
- 디지털 트윈

- 서비스형 소프트웨어
- 시맨틱 웹

### DB 신기술

### 회복 / 병행제어
```
: 데이터베이스가 손사오디었을 때 손상되기 이전의 정상 상태로 복구하는 작업을 가리킨다
```

#### REDO / UNDO
- REDO
```
: 디스크에 저장된 로그를 분석하여 트랜잭션의 시작과 완료에 대한 기록이 있는 트랜잭션들의 작업을 재작업한다
```

- UNDO
```
: 디스크에 저장된 로그를 분석하여 트랜잭션의 시작에 대한 기록은 있지만 완료 기록은 없는 트랜잭션에 대해 모두 취소한다
```

#### 즉각 갱신기법
```
: 트랜잭션이 데이터를 갱신하면 트랜잭션이 부분 완료되기 전이라도 즉시 실제 데이터베이스에 반영하는 방법
```

#### 로킹
```
: 액세스하기 전에 잠금을 요청해서 락이 허락되어야만 그 로킹 단위를 액세스할 수 있도록 하는 기법
```
- 로킹 단위
```
: 병행제어에서 한꺼번에 로킹할 수 있는 객체의 크기를 의미한다
```

#### 타임스탬프 순서
```
: 시간표를 부여하여 부여된 시간에 따라 트랝개션 작어블 수행하는 기법
```
