# 정보처리기사 - 제품 소프트웨어 패키징

해당 포스트에서는 정보처리기사 개념문제집 6장 **제품 소프트웨어 패키징**을 정리한다   

## 릴리즈 노트 작성 항목

- 머릿말
- 개요
- 목적
- 문제요약
- 재현 항목
- 수정/개선 내용
- 사용자 영향도
- SW 지원 영향도
- 노트
- 면책 조항
- 연락처

## 디지털 저작권 관리의 구성요소

- 클리어링 하우스
- 패키저
- 콘첸트 분배자
- 콘첸츠 소비자
- DRM 컨트롤러
- 보안 컨테이너

## 디지털 저작권 관리의 기술 요소

- 암호화
- 키 관리
- 암호화 파일 생성
- 식별 기술
- 저작권 표현
- 정책 관리
- 크랙 방지

## 소프트웨어 설치 매뉴얼
```
: 개발 초기에서부터 적용된 기준이나 사용자가 소프트웨어를 설치하는 과정에 필요한 내용을 기록한 설명서와 안내서이다
```

## 소프트웨어 패키징의 형상 관리
```
: 개발 과정에서 소프트웨어의 변경사항을 관리하기 위해 개발된 일련의 활동이다
```

## 형상 관리 기능

- 형상 식별
- 버전 제어
- 형상 통제
- 형상 검사
- 형상 기록

## 소프트웨어의 버전 등록 관련 주요 기능

- 저장소
- 가져오기
- 체크아웃
- 체크인
- 커밋
- 동기화

## 빌드 자동화 도구
```
: 빌드를 포함하여 테스틑 및 배ㅗ를 자동화하는 도구
```
