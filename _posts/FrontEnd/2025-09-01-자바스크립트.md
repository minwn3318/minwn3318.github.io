## 함수 생략
뚱뚱한 화살표를 활용하여 함수 표현 축약 가능하다
1. 이름이 없는 경우 **function** 키워드 생략 가능
```
const test = function() {}

const test = () => {}
```

2. **실행문이 출력문**일 경우 코드 블록 생략 가능
```
const test = function(x) {
    console.log(x)
}

const test = x => console.log(x);
```

3. 함수의 **인자값이 하나**인 경우 괄호도 생략 가능
```
const test = function(x) { }

const test = x => { }
```

4. **실행문이 반환문**일 경우 코드블록 생략 가능
```
const test = function(x){
    return x*x;
}

const test = x => x*x;
```
